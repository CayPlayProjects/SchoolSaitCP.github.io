## 📥 Часть 1. Скачивание и открытие HTML-файла

1. 🔍 Найдите в репозитории файл `index.html`.  
2. 🖱️ Кликните на него, чтобы открыть предпросмотр.  
3. В правом верхнем углу (возле имени файла) нажмите на **⋮ троеточие**.  
4. В выпадающем меню выберите **📥 «Скачать файл» / Download file**.  
5. ⏳ Дождитесь окончания загрузки на ваш компьютер.  
6. 📂 Перейдите в папку загрузок и откройте `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari и т.д.).

---

## 📦 Часть 2. Скачивание JSON-файла (бэкап данных сайта)

1. 🔙 Вернитесь в репозиторий.  
2. 🔍 Найдите файл **«бэкап данных сайта.json»** (или с похожим названием и расширением `.json`).  
3. 🖱️ Кликните на него, чтобы открыть предпросмотр.  
4. Нажмите на **⋮ троеточие** в правом верхнем углу.  
5. Выберите **📥 «Скачать файл» / Download file** и сохраните на компьютер.

---

## ♻️ Часть 3. Восстановление данных на сайте

1. ✅ Убедитесь, что сайт из файла `index.html` всё ещё открыт в браузере.  
2. В правом верхнем углу интерфейса найдите кнопку **💾 «Сохранить данные»**.  
3. Нажмите на неё — появится меню с двумя опциями.  
4. Выберите **📂 «Загрузить из файла»**.  
5. Нажмите **📎 «Выбрать файл»** — откроется окно выбора файлов.  
6. Найдите и выберите скачанный ранее **«бэкап данных сайта.json»**.  
7. Нажмите кнопку **🔄 «Восстановить данные»**.  
8. ⏳ Дождитесь завершения (обычно это занимает несколько секунд).

---

## 💡 Примечания и советы

- 🆕 **Если JSON-файл не нужен.**  
  Файл `.json` содержит изначальный шаблон данных. Если вы хотите начать с чистого листа — просто пропустите **Часть 2** и **Часть 3**.

- ✅ **Как проверить, что восстановление прошло успешно?**  
  После нажатия **«Восстановить данные»** интерфейс обновится и отобразит готовые разделы/темы из шаблона.

- ⚠️ **Проблемы с загрузкой?**  
  - Убедитесь, что вы выбрали именно файл `.json`.  
  - Проверьте, завершилась ли загрузка JSON-файла.  
  - Перезагрузите `index.html` и повторите попытку.
