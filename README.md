## Часть 1. Скачивание и открытие HTML‑файла

1. Найдите в репозитории файл `index.html`.
2. Кликните на `index.html`, чтобы открыть его предпросмотр.
3. В правом верхнем углу экрана (возле имени файла) нажмите на **троеточие** (`...`).
4. В выпадающем меню выберите опцию **«Скачать файл»** / **Download file**.
5. Дождитесь завершения загрузки файла на ваш компьютер.
6. Перейдите в папку загрузок и откройте файл `index.html` с помощью любого современного веб‑браузера (Chrome, Firefox, Edge, Safari и т. д.).


## Часть 2. Скачивание JSON‑файла (бэкап данных сайта)

1. Вернитесь в репозиторий.
2. Найдите файл с названием **«бэкап данных сайта.json»** (или похожим, с расширением `.json`).
3. Кликните на файл, чтобы открыть его предпросмотр.
4. Нажмите на **троеточие** в правом верхнем углу.
5. Выберите **«Скачать файл»** / **Download file** и сохраните файл на компьютер.

## Часть 3. Восстановление данных на сайте

1. Убедитесь, что у вас по‑прежнему открыт сайт из файла `index.html` в браузере.
2. На странице сайта найдите кнопку **«Сохранить данные»** — обычно она располагается в **правом верхнем углу** интерфейса.
3. Нажмите на кнопку **«Сохранить данные»**. Появится меню с двумя опциями.
4. Выберите кнопку **«Загрузить из файла»**.
5. Нажмите на кнопку **«Выбрать файл»**. Откроется стандартное окно выбора файлов на вашем компьютере.
6. Найдите и выберите скачанный ранее файл **«бэкап данных сайта.json»**.
7. После выбора файла нажмите кнопку **«Восстановить данные»**.
8. Дождитесь завершения процесса восстановления. Обычно это занимает несколько секунд.


## Примечания и советы

* **Если JSON‑файл не нужен.** Файл `.json` содержит изначальный шаблон данных. Если вы уже знакомы с функционалом сайта и хотите начать работу с нуля (создавать свои темы и т. д.), вы можете пропустить шаги 2–9 (всё, что связано с JSON‑файлом).
* **Проверка успешности восстановления.** После нажатия **«Восстановить данные»** интерфейс сайта должен обновиться и отобразить данные из JSON‑файла. Проверьте разделы, темы или элементы, которые должны были быть в шаблоне.
* **Проблемы с загрузкой.** Если кнопка **«Выбрать файл»** не реагирует или файл не загружается:
    * убедитесь, что вы выбрали именно файл с расширением `.json`;
    * проверьте, полностью ли завершился процесс скачивания JSON‑файла;
    * попробуйте перезагрузить страницу с `index.html` и повторить шаги с начала.
